## 如何使用？<br />How to use it?
1. 將程式碼封存檔下載下來並解壓縮
2. 將 C_CPP_project_template 目錄改名為您的專案名稱
3. 將上述目錄內的 .git 目錄（Git 版本控制系統的目錄）移除，將專案相關檔案改名為您的專案名稱
4. 開啟專案，於專案的屬性設定將專案的名稱改為您的專案名稱
5. Start coding :)
