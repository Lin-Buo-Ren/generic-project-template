## 如何用此軟體建構解決方案建構軟體？
1. 在終端機中將當前工作目錄(current working directory)切換到本目錄中
2. 執行 cmake ../.. 命令
3. 執行 make 命令
4. 建構出來的檔案會在當前工作目錄中的 Built software/ 目錄底下
